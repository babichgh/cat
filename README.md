# cat
Helps to quickly see file content or write something down on it
## Prerequisites
- g++ compiler
## Installation
1. Clone the repository
   ```
   git clone https://github.com/babichgh/cat.git
   cd cat
   ```
2. Compile
   ```
   make
   ```
3. Run
   ```
   make run
   ```
## Project structure
